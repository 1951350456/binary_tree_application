# binary_tree_application
<1> 设计一个二叉树系统，除包含二叉树的相关运算功能外，还要实现以下功能： 
    （1）按中序次序输出二叉树中各结点的值及其所对应的层次数；
               例如：构建如图所示的二叉树，输出结果：
                                D:3,  B:2,  E: 3,  A: 1  F: 3,  C:2 G:3 

  
    （2）实现三种次序的线索化算法；
    （3）在线索二叉树中指定位置上插入一个用户自定义的结点；
              要求：给出测试案例验证相关运算求解结果。
